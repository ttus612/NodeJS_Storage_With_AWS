# 😍Trình bày về project:
## 📚Sử dụng Node JS và dùng AWS để lưu trữ
- "git clone" clone repo này về.
- Sau đó gõ "npm i" vào terminal
- Tạo file .env Ở root folder và cấu hình cho nó với các nội dung sau:
  + ACCESS_KEY_ID=
  + SECRET_ACCESS_KEY=
  + REGION=
  + DYNAMODB_TABLE_NAME=
  + S3_BUCKET_NAME=
- Tất cả các mục trên ta phải tạo trên AWS trước
